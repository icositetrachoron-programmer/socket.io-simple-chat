# socket.io-simple-chat

A minimal live chat app made with [socket.io](https://socket.io)

## Running

Make sure you have Express and Socket.io installed:
```
npm install express
npm install socket.io
```
then run server.js:
```
node server.js
```
and go to `localhost:3000`:
